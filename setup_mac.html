Installing on Mac OSX

Using the terminal

It is possible to install Python and get everything set up to use a graphical interface, but it turns out that for Python on Mac OSX the command-line way of doing things is much simpler.

If you've never used a terminal before, check out these instructions.

Install homebrew

Install homebrew following the (very simple) instructions from brew.sh.

Once homebrew is installed run:

Install Python and the botchallenge module

If you don't already have Python 3.4 installed, then run:

brew install python3

If you have already installed Python 3 but it's an older version than 3.4, upgrade it with:

brew upgrade python3

Instaling Python3 through homebrew will also install pip3, use it to install the botchallenge module like this:

pip3 install botchallenge

Editing and running Python scripts

To run an interactive python interpreter, run:

python3

To run a python script that you've written to a file (where the file in this case is called script.py) run:

python3 script.py

You can use any text editor to edit the Python files, then save the file and come back to the terminal to run it. I recommend TextWrangler as a good free text editor which works well for Python.
