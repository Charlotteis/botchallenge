The Bot Challenge

Remove the boring bits of Minecraft with your own Python code.

Installing on Windows

You'll need to download and install Python (if you don't have it already) then install two extra modules.

Python versions

If you already have Python installed, check what version it is. To check the version, open IDLE and the first thing it says is Python then the version number. If it's less than 3.3 then you'll need to install the new version.

If you have Python 3.3 then it'll work, you just need to install asyncio from here.

If you don't have Python installed or it's an old version, install Python 3.4 from here.

Installing Greenlet

Download and run the greenlet installer from here.

If you are using Python3.3 or 64-bit, then find the matching .msi installer from this list of greenlet module installers.

Installing BotChallenge

Download and run the botchallenge installer from here.

Aagain, for other versions choose the .msi installer from this list of botchallenge module installers.

Next steps

As part of installing Python, you should have The IDLE Python editor installed. Open it up, create a new file (File > New) and save it as mybot.py (you can call it what you like as long as it has .py on the end). Then type in (or copy-paste) the following:

from botchallenge import *

The go to Run > Run Module to run it (or hit F5).

If it prints nothing, congratulations! Everything is installed correctly.

If it shows an error, then there might have been a problem installing the modules or you might have more than one version of Python installed and you're using the wrong one.

If it's working, connect to the server and get started.
