The Bot Challenge

Remove the boring bits of Minecraft with your own Python code.

Welcome to the Bot Challenge.

Minecraft is a great example of how in everyday life we have to do some boring repetitive tasks, something a robot could do. Since our world and jobs are going to gradually be replaced with robots and computers, learning to take our every day tasks and break them down into step-by-step computer programs is going to be a valuable skill. Here's your chance to play Minecraft and practice your Python coding at the same time.

What you need

  1. A little bit of Python experience (just the basics)
  2. A regular Minecraft account (from http://minecraft.net)
  3. A computer with Minecraft installed and an internet connection

Note that this is not a modification to your Minecraft client, we've modified the server only so you don't need to mess around with mods unless you want to run your own server. You'll only need to install standard Python and a couple of extra Python modules, but we've got instructions for that below.

How it works

When you join our multiplayer Minecraft server which has the BotChallenge plugin installed, it looks just like a regular shared Minecraft world. The difference is, you can spawn a robot helper. It looks like a regular Pumpkin, but you can connect to it from your Python code and give it commands, you can tell it to move around (it can fly), mine and place blocks.

We've got a bunch of example scripts to get you started, like a bot which just follows you around, a bot which mines stairs and a bot which builds a simple dirt hut to get you started. See the example scripts.

Setting up

First, you need to connect to our Minecraft server to spawn your bot. Since we're still testing this out we're keeping the address a secret so if you've got the address in your email or someone shared it with you, then go ahead and join the server in Minecraft as a regular player with a regular client.

When you're there (in Minecraft), type:

/spawnrobot

Then right-click anywhere and you should see your brand new Pumpkin minion appear! It's there waiting for your instructions. You can give a few instructions in the game itself, by typing: 

/robot move up

You can use any of up, down, left, right, forward, backward, north, south, east or west for the direction. You can also tell the robot to 'turn' to face a direction or 'mine' the block in any direction. We chose a pumpkin to represent the bot so that you can see which direction is it's facing.

Of course running these commands one by one is even more tedious than mining by yourself, so now that our robot exists, let's start with a simple script.

Before this will work, you need to install the botchallenge library, instructions for different platforms are below. Once you've got it setup, open a Python file and past this simple script:

from botchallenge import *
robot = Robot()

for i in range(10):
    robot.move(Dir.UP)
    robot.place(Dir.DOWN, BlockType.DIRT)
    robot.move(Dir.RIGHT)
    robot.place(Dir.DOWN, BlockType.DIRT)

If there's enough space nearby, your robot should be building you a some dirt stairs! If that works, then you're good to go, you should check out the example scripts for more ideas of what you can do.

Installing on Mac

First install Python3, If you like terminals I would recommend using homebrew from http://brew.sh Once homebrew is installed run:

brew install python3

This will also install pip3, use it to install the botchallenge module.

pip3 install botchallenge

If you don't want to use homebrew, then you could install Python3 from the python.org site, then follow the instructions at http://pip.pypa.io for installing pip, then use pip3 to install botchallenge as above.

Installing on Linux

On Ubuntu/Debian run these commands, for other distributions there should be equivalent packages.

sudo apt-get install python3 python3-dev python3-pip

Then use pip3 to install the botchallenge module

sudo pip3 install botchallenge

Installing on Windows

You'll need to download and install each of these packages (Python must be installed before you install the others).

  1. Python 3.3 or higher: https://www.python.org/download
  2. The Greenlet Python module: https://pypi.python.org/pypi/greenlet (Make sure you pick the same version as the Python version you installed)
  3. The botchallenge module: https://pypi.python.org/pypi/botchallenge

As part of installing Python, you should have The IDLE Python editor installed. Open it up, create a new file (File > New) and save it as mybot.py (you can call it what you like as long as it has .py on the end). Then type in (or copy-paste) the following:

from botchallenge import *

The go to Run Module (in the menu) to run it (or hit F5). If it prints nothing, then everything is installed correctly. If it shows an error, then there might have been a problem installing the modules or you might have more than one version of Python installed and you're using the wrong one.

If it's working, you can try the simple bot above.
